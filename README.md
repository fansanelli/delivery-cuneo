- Dati OSM

I dati sono ricavati con query overpass:

```
( area[name=Cuneo]; )->.searchArea;
(
  node["delivery:covid19"=yes](area.searchArea);
  way["delivery:covid19"=yes](area.searchArea);>;

);
out body;

```

l'output va salvato come:

```
delivery.osm
```
